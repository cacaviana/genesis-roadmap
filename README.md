# Genesis Roadmap

Plano de ação visual do Genesis: arquitetura atual, arquitetura alvo (multi-canal IT Valley com messaging-service da Polly), 5 fases de migração e ferramentas do ecossistema.

Construído com **Svelte 5 + Vite + TypeScript**, deploy no **Azure Static Web Apps** (NoPROD).

## Rodar local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Estilo: PAR adapted (dark navy `#0F172A` + purple `#A855F7`).
